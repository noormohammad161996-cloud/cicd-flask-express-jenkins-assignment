# Express Frontend

This is a simple Express.js frontend application developed for the CI/CD Deployment Assignment.

## Project Structure

```
express-app/
├── app.js
├── package.json
└── README.md
```

## Prerequisites

- Node.js
- npm

## Install Dependencies

```bash
npm install
```

## Run the Application

```bash
npm start
```

## Test the Application

Open your browser:

```
http://localhost:3000
```

Or use curl:

```bash
curl http://localhost:3000
```

Expected Output:

```
Hello from Express Frontend - CI/CD Assignment
```
