# CI/CD Flask & Express Jenkins Assignment

## Project Overview

This project demonstrates a complete CI/CD workflow for Flask and Express applications using Jenkins on AWS EC2.

## Technologies Used

- AWS EC2
- Ubuntu 26.04
- Git & GitHub
- Jenkins
- Python 3
- Flask
- Node.js
- Express.js
- PM2
